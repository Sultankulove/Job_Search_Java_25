Запрос GET должен возвращать один из следующих кодов состояния HTTP: 200 OK, 204 No Content (нет содержимого), 404 (не найдено).

Если клиент пытается создать собственный URI и выдает запрос POST для определенного URI, сервер возвращает код состояния HTTP 400 (BAD REQUEST), чтобы указать, что метод не поддерживается.

Запрос POST должен возвращать один из следующих кодов состояния HTTP: 200 OK, 201 Создано, 204 No Content (нет содержимого), 400 (недопустимый запрос), 405 (метод не разрешен).

Запрос PUT должен возвращать один из следующих кодов состояния HTTP: 200 OK, 201 Создано, 204 No Content (нет содержимого), 409 (конфликт)

Запрос PATCH должен возвращать один из следующих кодов состояния HTTP: 200 OK, 400 (недопустимый запрос), 409 (конфликт), 415 (неподдерживаемый тип носителя).

Запрос DELETE должен возвращать один из следующих кодов состояния HTTP: 204 (НЕТ СОДЕРЖИМОГО), 404 (НЕ НАЙДЕНО).
